
# Game Mata Mosquito

Um projeto desenvolvido para fins de fixação de conhecimentos.




## Do que se trata o projeto

Um jogo simples que roda no próprio browser.

Seu objetivo e eliminar todos os mosquitos que aparecerem em sua tela em um determinado período de tempo.

* Você possui níveis de dificuldade onde poderá escolher

* Possui três vidas que poderá gastar durante o jogo.
